# WeatherPulse Pro - Advanced Weather Intelligence

## Project Overview

WeatherPulse Pro is an advanced weather intelligence dashboard that provides real-time weather data, forecasts, and a wide range of meteorological insights. It is built with modern web technologies including HTML5, CSS3, and vanilla JavaScript, focusing on a clean, intuitive user interface and robust functionality. The application leverages the Open-Meteo API to deliver accurate and free weather information without requiring an API key.

## Features

- **Real-Time Weather Data**: Get up-to-the-minute weather information for any location worldwide.
- **Comprehensive Forecasts**: View detailed hourly and daily weather forecasts.
- **Global Location Search**: Easily search for any city or location.
- **Geolocation Support**: Automatically fetch weather data for your current location.
- **Air Quality Index (AQI)**: Monitor air quality with detailed pollutant information.
- **Advanced Weather Metrics**: Access a wide range of data including humidity, wind speed and direction, pressure, UV index, visibility, and more.
- **Performance-Optimized Caching**: An intelligent, persistent caching system reduces API calls and speeds up data loading.
- **Auto-Refresh Functionality**: Keep your weather data current with automatic updates at configurable intervals (1, 5, 10, 15, 30, 60 minutes, or custom).
- **Manual Refresh**: Instantly update the weather data with a single click.
- **Customizable Settings**: Personalize your experience with options for temperature units (Celsius, Fahrenheit), wind speed units, and more.
- **Favorites & Recent Searches**: Save your favorite locations for quick access and view your recent search history.

## Setup Instructions

To run this project locally, follow these simple steps:

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd WeatherPulse
    ```
3.  **Open the application**:
    Simply open the `index.html` file in your favorite web browser. No build steps or dependencies are required.

## API Used

This project is powered by the **Open-Meteo API**, a free and open-source weather API.

- **Website**: [https://open-meteo.com/](https://open-meteo.com/)
- **Key Features**: No API key is required, and it offers a generous free tier with a wide array of weather data.
- **Documentation**: [https://open-meteo.com/en/docs](https://open-meteo.com/en/docs)
